Blogging Platform API - Endpoints

  • GET /posts/ Get all blog posts
  • GET /posts/<id>/ Get single blog post by ID
  • POST /posts/ (create new post - use API client like Postman)
  • PUT /posts/<id>/ (update post by ID - use API client)
  • DELETE /posts/<id>/ (delete post by ID - use API client)
  • GET /posts/search/<term>/ Search posts by term
